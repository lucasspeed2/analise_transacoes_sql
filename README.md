# 📊 Análise de Transações Financeiras com SQL

Desenvolvi esse projeto utilizando **MySQL** para análise de dados de transações financeiras.

O objetivo deste projeto é praticar **consultas SQL aplicadas a cenários reais de análise de dados**, identificando padrões, inconsistências e comportamento de clientes.

---

# 🎯 Objetivos do Projeto

- Praticar SQL intermediário
- Utilizar CASE WHEN
- Aplicar GROUP BY
- Utilizar Window Functions
- Identificar possíveis erros nos dados financeiros

---

# 📊 Análises Realizadas

O projeto tem várias consultas SQL para análise de transações:

- Identificação de valores negativos
- Classificação de transações de alto valor
- Total de crédito e débito por cliente
- Cálculo de saldo líquido
- Identificação de múltiplas transações no mesmo dia
- Ranking de clientes com maior movimentação
- Detecção de valores fora do padrão
